Dokumentasi Endpoint Api

Menggunakan Endpoint GET /api

Anda dapat menggunakan endpoint ini untuk mengecek apakah situs online / offline berdasarkan url tanpa https:// atau http://. Gunakan parameter query url untuk menentukan url website yang ingin di cek.

Contoh penggunaan:

https://webdowncheck.api.akuari.my.id/api?url=google.com

Respons:

{
  "success": true,
  "website": "google.com",
  "response": "Website is Up (Online)",
  "date": "Selasa, 26 September 2023 pukul 9.48 PM WIB"
}

Catatan: Pastikan untuk selalu menggunakan url yang valid untuk mengecek apakah situs offline / online.
